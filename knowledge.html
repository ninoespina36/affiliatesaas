This front side is what your Affiliates members see.

It can be fully customized using our script builder on the left menu.

To edit the front side you have two options:
  • Edit our templates, go to Global setting Front side, and view the front side template (default and landing).
  • Edit front side using site builder, go to Script builders Site builder and create your own theme using site builder.

After you create your theme or edit your theme in both options, so go to Global setting Configuration Site settings and choose your theme for front side and save.
